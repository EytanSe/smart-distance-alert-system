# מערכת התרעת מרחק חכמה 🚗📏
### Smart Distance Alert System

פרויקט גמר בקורס **מערכות משובצות** — מערכת בסגנון חיישן חניה המבוססת על Arduino.
המערכת מודדת ברציפות את המרחק לעצם ומספקת התרעה חזותית (3 נוריות) וקולית (זמזם
המאיץ ככל שמתקרבים).

---

## ✨ תכונות עיקריות
- 📡 מדידת מרחק בעזרת חיישן אולטרסוני **HC-SR04** (טווח 2–400 ס"מ).
- 🟢🟡🔴 שלוש נוריות הנדלקות בהדרגה לפי אזור המרחק.
- 🔊 זמזם פסיבי המצפצף **מהר יותר ככל שהעצם קרוב יותר**.
- ⚡ תכנון **לא-חוסם** (non-blocking): ללא `delay()` בזרימה הראשית — מבוסס
  `millis()`, פסיקת **Timer1** ופסיקת **ECHO** חיצונית.
- 🔌 הפעלת הזמזם דרך טרנזיסטור **BC547 (NPN)** המשמש כמתג.

## 🧩 אזורי ההתרעה

| מרחק | נוריות | זמזם |
|------|---------|-------|
| מעל 50 ס"מ | 🟢 ירוקה | שקט |
| 20–50 ס"מ | 🟢🟡 ירוקה + צהובה | צפצוף איטי ומואץ |
| מתחת ל-20 ס"מ | 🟢🟡🔴 הכול דולק | צפצוף מהיר |

---

## 🔧 חיווט מהיר (Pin Map)

| רגל Arduino | רכיב |
|--------------|-------|
| D4 | HC-SR04 TRIG |
| D2 | HC-SR04 ECHO (פסיקה INT0) |
| D11 | נורית ירוקה (דרך 220Ω) |
| D10 | נורית צהובה (דרך 220Ω) |
| D9 | נורית אדומה (דרך 220Ω) |
| D6 | בסיס BC547 (דרך 1kΩ) → זמזם |
| 5V / GND | אספקה ואדמה משותפת |

> תיאור מלא של כל חוט נמצא בקובץ [docs/wiring_diagram.md](docs/wiring_diagram.md).

## 🚀 איך מריצים

1. פתחו את [code/distance_alert.ino](code/distance_alert.ino) ב-**Arduino IDE**.
2. חברו את החומרה לפי [docs/wiring_diagram.md](docs/wiring_diagram.md).
3. בחרו לוח **Arduino Uno** ואת היציאה (COM) המתאימה.
4. לחצו **Upload**.
5. פתחו את **Serial Monitor** (9600 baud) כדי לראות את המרחק הנמדד.
6. הזיזו חפץ לעבר החיישן — הנוריות והזמזם יגיבו בזמן אמת.
