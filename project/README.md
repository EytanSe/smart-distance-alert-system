# מערכת התרעת טמפרטורה חכמה 🌡️🔔
### Smart Temperature Alert System

פרויקט גמר בקורס **מערכות משובצות** — מערכת התרעה על התחממות יתר המבוססת על Arduino.
המערכת מודדת ברציפות את הטמפרטורה ומספקת התרעה חזותית (3 נוריות) וקולית (זמזם
המאיץ ככל שחם יותר).

> **רקע:** הפרויקט תוכנן במקור כ"מערכת התרעת מרחק" עם חיישן אולטרסוני HC-SR04, אך
> חיישן זה אינו כלול בערכת **Arduino Upgraded Learning Kit** שברשותנו. המערכת
> הותאמה לחיישן הטמפרטורה **LM35DZ** הקיים בערכה — תוך שמירה על אותה ארכיטקטורה:
> חיישן → שלושה אזורי התרעה (נוריות) → זמזם המאיץ.

---

## ✨ תכונות עיקריות
- 🌡️ מדידת טמפרטורה בעזרת חיישן אנלוגי **LM35DZ** (10 mV/°C, טווח 0–100°C).
- 🟢🟡🔴 שלוש נוריות הנדלקות בהדרגה לפי אזור הטמפרטורה.
- 🔔 זמזם המצפצף **מהר יותר ככל שחם יותר**.
- ⚡ תכנון **לא-חוסם** (non-blocking): ללא `delay()` בזרימה הראשית — מבוסס
  `millis()` ופסיקת חומרה **Timer1**.
- 🔧 קוד התומך גם בזמזם פסיבי וגם באקטיבי (מתג `BUZZER_IS_PASSIVE`).
- 🧰 **כל הרכיבים מתוך הערכה** — לא נדרשת רכישה נוספת.

---

## 🧩 אזורי ההתרעה

| טמפרטורה | נוריות | זמזם |
|------|---------|-------|
| מתחת ל-30°C | 🟢 ירוקה | שקט |
| 30–40°C | 🟢🟡 ירוקה + צהובה | צפצוף איטי ומואץ |
| מעל 40°C | 🟢🟡🔴 הכול דולק | צפצוף מהיר |

> הספים ניתנים לשינוי בקלות בקוד (`TEMP_WARM`, `TEMP_HOT`).

## 🔧 חיווט מהיר (Pin Map)

| רגל Arduino | רכיב |
|--------------|-------|
| A0 | LM35DZ — Vout (יציאה אנלוגית) |
| D11 | נורית ירוקה (דרך 220Ω) |
| D10 | נורית צהובה (דרך 220Ω) |
| D9 | נורית אדומה (דרך 220Ω) |
| D6 | זמזם (הפעלה ישירה) |
| 5V / GND | אספקה ואדמה משותפת (כולל +Vs ו-GND של ה-LM35) |

> תיאור מלא של כל חוט נמצא בקובץ [docs/wiring_diagram.md](docs/wiring_diagram.md).

## 🚀 איך מריצים

1. פתחו את [code/temperature_alert.ino](code/temperature_alert.ino) ב-**Arduino IDE**.
2. חברו את החומרה לפי [docs/wiring_diagram.md](docs/wiring_diagram.md).
3. (מומלץ) בדקו את ה-LM35 עם המולטימטר: ~0.25V בטמפרטורת חדר.
4. בחרו לוח **Arduino Uno** ואת היציאה (COM) המתאימה.
5. אם הזמזם **אקטיבי** — שנו בקוד `#define BUZZER_IS_PASSIVE 0`.
6. לחצו **Upload**.
7. פתחו את **Serial Monitor** (9600 baud) כדי לראות את הטמפרטורה הנמדדת.
8. געו בחיישן או קרבו מקור חום — הנוריות והזמזם יגיבו בזמן אמת.
